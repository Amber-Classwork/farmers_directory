Welcome to the Farmers Directory API

API Endpoints

The endpoints of the API starts at the base route:

  • /users

    • Has full CRUD functionality for managing the users

  • /farmers

    • Has full CRUD functionality for managing the farmers. farmers would created at this endpoint as well/ registered.

  • /products

    • Full CRUD functionality for managing products

  • /authenticate

    • Logs in the user and is incharge of authorizing access to the API.

API Requirements:

This API was created with the following requirements in mind

  • Allow the user to create a new account
  • Allow the users to view products offered by farmers.
  • Allow users to view farmers that offers the products so they can compare products and distance from farmers
  • Allow farmers to signup and display their products and contact information
